# Formulate Overview
A form builder for Umbraco.

# Contributing
Requires:
* Visual Studio 2013
* Node.js
* npm
* grunt-cli (installed globally)

# Building
These are the steps you can take to build and test Formulate:
* Build the solution.
* Run grunt.
* Run the sample website.
